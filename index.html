🔗 Connect Google Sheets
One-time setup • takes ~5 minutes • free forever
Step 1 — Create your Google Sheet
Go to sheets.new and create a new spreadsheet.
Name it Avero Activity Log
Step 2 — Open Apps Script
In the sheet: Extensions → Apps Script
Delete any existing code and paste the script below, then click Deploy → New Deployment → Web App.
Set Execute as: Me and Who has access: Anyone. Copy the deployment URL.
function doPost(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var data = JSON.parse(e.postData.contents);
  var sheetName = data.sheet || 'Activity';
  var sheet = ss.getSheetByName(sheetName) || ss.insertSheet(sheetName);
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(data.headers || Object.keys(data.row));
  }
  sheet.appendRow(Object.values(data.row));
  return ContentService.createTextOutput('ok');
}
function doGet(e) {
  return ContentService.createTextOutput('Avero Logger Ready');
}

🌿 COIR Substrate Classifications
Types and grades you need to know

Pure Coir Pith

High water retention. Best for tomato, cucumber. Fine particle size. EC must be very low.

Coir Chips

High air porosity. Excellent for orchids, cannabis, heavy feeders needing drainage.

50/50 Mix

Balanced water/air ratio. Most versatile. Most popular for commercial production worldwide.

70/30 Pith/Chip

Standard tomato/pepper profile. Good water retention with adequate drainage.

30/70 Pith/Chip

High drainage. Cannabis, strawberry, orchids. Better oxygenation at roots.

Buffered Coir

Pre-washed to remove salts. Essential for hydroponic drip systems. Required for EU markets.
